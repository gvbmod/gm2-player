# Gvbvdxx Mod 2 - Project Player.

An attempt to replace the current GvbvdxxMod2HTML with a better one.
This compiles like scratch-gui, make sure you link GvbvdxxMod2's scratch-vm with this for it to work.
This attempts to get the accuratest results with scratch-gui.

Support with most TurboWarp extensions (with working overlay features!).
